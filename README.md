# Mobile Sales Data Cleaning using SQL

## Project Overview
This project focuses on cleaning and preprocessing raw mobile sales data using SQL. The dataset contained missing values, duplicate records, invalid entries, and inconsistent formats. The goal was to transform the raw dataset into a clean and analysis-ready dataset.

---

## Dataset Information
The dataset contains:
- Sale ID
- Mobile Model
- Brand
- Price
- Quantity
- Sale Date
- Customer Name
- Payment Method
- City
- Rating

---

## Tools Used
- MySQL
- SQL

---

## Data Cleaning Tasks Performed
- Checked and handled null values
- Identified duplicate records
- Removed invalid price and quantity values
- Standardized brand names
- Standardized city names
- Cleaned customer names
- Converted date formats
- Created a cleaned table

---

- Aggregate Functions
